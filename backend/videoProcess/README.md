# Movilidad-y-Tranporte
Desarrollo para el proyecto de Analisis Dinamico de la Movilidad y Transporte 

Es necesario instalar las siguientes librerías para poder ejecutar de manera correcta el software, además de ya contar con Python3 instalado.
  Las librerías son: 
  Ultralytics
  PyTorch
  Roboflow
  Supervision
  torchvision

Para el uso de gráficas Nvidia es necesario tener Nvidia Toolkit
